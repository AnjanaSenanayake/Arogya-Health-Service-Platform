# Arogya Web App
Arogya is a platform for personalized medical assitance and information

### Installation

Arogya requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and start the server + front-end.

```sh
$ npm install
$ cd backend
$ npm install
$ cd ../frontend
$ npm install
$ cd ..
$ npm start
```
